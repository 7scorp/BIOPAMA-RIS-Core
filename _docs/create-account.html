--- title: Create an account icon: fa-book order: 1 ---
  • Click Login
  • Click Signup
  • Fill in the form that appears

Things worth noting when creating a new account:

  • Users can immediately add content with an account, but it remains in "Draft" mode (hidden to the public)
  • There are 2 ways to publish content so it is publicly accessible: the new user account must be approved, or the content can be published by BIOPAMA personnel.
